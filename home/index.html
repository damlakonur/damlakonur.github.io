Projects

Generative Model of 3D Heads

As part of the 3D Scanning & Spatial Learning practical course at TUM, I worked on a generative framework for reconstructing 3D human heads from sparse-view inputs. The model uses an auto-decoder to learn a latent distribution over human heads, trained on the CAFCA dataset.

The learned prior is leveraged to reconstruct high-quality head geometry using only a few images as input. Key tools include Python, PyTorch, and 3D Gaussian Splatting (3DGS).

This project combines ideas from 3D Gaussian Splatting and novel view synthesis, and serves as a strong foundation for learning-based 3D head avatar generation.

Bundle Adjustment

As part of the 3D Scanning & Motion Capture course at TUM, I implemented a Structure-from-Motion (SfM) pipeline from scratch in C++.

This project focused on:

  • Camera pose estimation, triangulation, and feature matching
  • Building a bundle adjustment module using Ceres Solver and Eigen
  • Creating a complete pipeline for sparse 3D reconstruction from unordered images

The project strengthened my understanding of multi-view geometry and optimization in 3D vision, with practical experience in low-level system design and numerical libraries.

Vision-Based Trajectory Control with MIT Racecar

This project focused on implementing vision-based trajectory control for the MIT Racecar platform.

Key contributions:

  • Used a ZED stereo camera to capture real-time road imagery, processed using Python and the OpenCV library.
  • Computed the lateral error of the vehicle relative to the lane center in the body frame using image processing techniques.
  • Designed and implemented multiple controller structures for lateral and longitudinal control, including:
    • Proportional (P) Controller
    • Proportional-Derivative (PD) Controller
    • Stanley Controller

This work demonstrated an integrated approach to autonomous driving using computer vision and feedback control, all tested in real-time on a physical platform.
